Avatars

Avatars are user profile pictures, usually found in circular shapes. Avatars can be used for 'container' objects — projects, spaces, repositories, etcetera — to give them a recognizable visual identity. These representations are used for the attention grabbing on various social medias too.

Profile Avatars

These Avatars are used as profile images and can be used at several places too.

Squared Avatars

PISTON UI includes soft cornered square avatars which makes feel good and give you options to choose from size too.
